# Bootstrap UI for Sympa

A responsive UI for [Sympa](http://www.sympa.org) - mailing list manager - using [Twitter Bootstrap 3](http://getbootstrap.com).

## Installation

Make `fluid` directory in the `static_content` directory of sympa
Copy the bootstrap distribution (css and js) in `static_content/fluid/bootstrap-3.3.4`
Put tt2 files in the `etc/web_tt2` directory of sympa
